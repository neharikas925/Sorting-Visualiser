# Sorting Visualizer
A *Vue.js* web-app which visualizes - Bubble Sort, Selection Sort, and Insertion Sort algorithms.
## Live Demo
http://sortvisualiser.herokuapp.com/

## Project setup
```
npm install
```

npm run serve
